# Sovereignty Subsystem

Experimental reference implementation for an AI sovereignty control plane inside CODEX.

Canonical authority remains `wiredchaos/AGENTROPOLIS-SOVEREIGNTY`; CODEX is an incubation lane only and does not redefine doctrine or become the canonical policy authority.

Run:
- `pnpm sovereignty:typecheck`
- `pnpm sovereignty:test`
- `pnpm sovereignty:validate-registries`

The subsystem is designed to be extractable. Do not introduce CODEX-specific path dependencies into application code.
